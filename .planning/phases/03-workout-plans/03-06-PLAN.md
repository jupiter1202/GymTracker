package de.jupiter1202.gymtracker.feature.plans

fun TemplatePreviewScreen(

    onImported: (Long) -> Unit
)

1. Obtain `WorkoutPlanViewModel` via `koinViewModel()` and `ExerciseViewModel` via `koinViewModel()`.
2. Collect `templates` and `exercises` StateFlows.
3. Derive: `val program = templates.find { it.id == templateId }`. If null, call `LaunchedEffect(Unit) { onNavigateBack() }` and return early.
4. Build exercise lookup map: `val exerciseLookup = exercises.associate { it.name.lowercase() to it.id }`.

**TopAppBar:**
- Title: program.name
- Navigation icon: back arrow

**Body (LazyColumn):**
- Description item: Text(program.description, style = bodyMedium, color = onSurfaceVariant, modifier = Modifier.padding(horizontal=16.dp, vertical=8.dp))
- Section divider with "Exercises" label
- For each day in program.days, show:
  - stickyHeader: Text(day.name, style = labelLarge, color = primary) — acts as day label
  - For each exercise in day.exercises, show a ListItem:
    - headlineContent: Text(exercise.exerciseName)
    - trailingContent: Text("${exercise.targetSets} × ${exercise.targetReps}", style = bodySmall)
- Spacer at bottom for button clearance (80.dp)

**Bottom sticky button:**
- Use `Box(Modifier.fillMaxSize())` + inner `Button` anchored to `Alignment.BottomCenter` with padding, OR use Scaffold `bottomBar`:
  - Scaffold bottomBar: `BottomAppBar` with full-width `Button("Use this program")` inside
  - On click: show loading indicator briefly, call:
    ```kotlin
    viewModel.importTemplate(program, exerciseLookup) { planId ->
        onImported(planId)
    }
    ```
- Disable button while import is in progress (use `var importing by remember { mutableStateOf(false) }`; set to true on click, ViewModel's onImported callback fires on the same coroutine scope so the screen navigates away before the user sees it disabled for long).

**Loading state for templates:** If `templates.isEmpty()`, show `CircularProgressIndicator` centered — this handles the brief moment before `loadTemplates()` completes in the ViewModel's init block.

```kotlin
composable(
    route = "template_preview/{templateId}",
    arguments = listOf(navArgument("templateId") { type = NavType.StringType })
) { backStackEntry ->
    val templateId = backStackEntry.arguments?.getString("templateId") ?: return@composable
    TemplatePreviewScreen(
        templateId = templateId,
        onNavigateBack = { navController.popBackStack() },
        onImported = { planId ->
            // Pop template_preview off back stack, then navigate to plan detail
            navController.popBackStack()
            navController.navigate("plan_detail/$planId")
        }
    )
}
```
