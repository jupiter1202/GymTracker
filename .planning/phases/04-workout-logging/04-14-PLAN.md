    ```kotlin
    /**
     * Mark an exercise as done and start rest timer
     * Called when user taps "Done" button without logging another set
     */
    fun markExerciseDone(exerciseId: Long) {
        viewModelScope.launch {
            // Get rest timer duration from settings
            val restTimerDuration = settingsRepository.restTimerSeconds
            restTimerDuration.collect { duration ->
                startRestTimer(duration)
            }
        }
    }
    ```
    
    Rationale:
    - Separate from logSet() — doesn't validate weight/reps
    - Retrieves rest timer duration from settings (same as logSet)
    - Starts rest timer immediately
    - Can be called multiple times per exercise (timer just restarts)
    - Simpler than logSet (no database persistence, no input validation)
  </action>
