## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-05-01 | Tampering | SettingsScreen rest timer input | mitigate | `toIntOrNull()` guard in onValueChange; `parsed in 10..600` range check before calling viewModel — only valid values reach SettingsRepository; SettingsRepository has `require(seconds in 10..600)` as second line of defense |
| T-04-05-02 | Denial of Service | SoundPool.play() on timer completion | accept | SoundPool plays a ~0.5s clip; no infinite loop risk; `soundLoaded` flag prevents play() if asset failed to load |
| T-04-05-03 | Information Disclosure | WorkoutSummary volume calculation | accept | Summary data is the user's own workout data; displayed only to the user; no network transmission |
</threat_model>
