## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-06-01 | Tampering | active_workout/{sessionId} route argument | accept | sessionId is a Long extracted via `NavType.LongType` — Navigation validates type; value comes from Room insert return value (server-side ID), not from user input directly |
| T-04-06-02 | Denial of Service | Crash recovery LaunchedEffect on every composition | mitigate | `LaunchedEffect(Unit)` runs once per composition entry (key = Unit) — not on every recomposition; runs once on app start |
| T-04-06-03 | Tampering | Post-hoc startedAt from DatePickerDialog | accept | startedAt is epoch ms from Android's DatePickerState; user picks a date for their own workout history — no cross-user impact; stored as Long in Room |
</threat_model>
